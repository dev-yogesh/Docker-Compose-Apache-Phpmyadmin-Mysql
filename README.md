# Docker-Compose for Php Based hosting

> Replacement for xampp/wampp
 
#### simple docker compose for php based hosting with phpmyadmin and mysql.


#### Images used

- php:8.0-rc-apache
- mysql:8.0
- phpmyadmin/phpmyadmin

#### Ports

- mysql db -> 3306
- phpmyadmin -> 8000
- apache www folder -> 8001
